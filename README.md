# RIT Next.js V2

Professional Next.js website for Rasel Innovation & Technology (RIT).

## Local run

```bash
npm install
npm run dev
```

## GitHub/Vercel

Upload all project files to the repository root. Vercel will detect Next.js automatically.

Before upload, remove the old static website files:
- index.html
- styles.css
- script.js

Keep the new folders and files:
- app/
- components/
- public/
- package.json
- tsconfig.json
- next-env.d.ts
- .gitignore
- README.md
