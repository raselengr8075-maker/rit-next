# Custom domain setup
1. Buy a domain from a registrar or Vercel.
2. Open Vercel Project → Settings → Domains.
3. Add the root domain and optional `www` domain.
4. Apply Vercel's DNS records and select one primary domain.
5. Set `NEXT_PUBLIC_SITE_URL`, redeploy, and verify HTTPS, canonical URLs, sitemap and robots.
6. Configure separate MX records when domain email is required.
