# SEO guide

Set `NEXT_PUBLIC_SITE_URL` to the final HTTPS origin without a trailing slash. The site generates `robots.txt`, `sitemap.xml`, canonical URLs, Open Graph/Twitter metadata, and structured data for the organization, website, breadcrumbs, articles, software applications, services, founder, and gallery.

Before publishing a record:

1. Use one descriptive English title and a faithful Bangla title.
2. Write a unique summary and SEO description supported by page content.
3. Use a stable lowercase slug; avoid changing a published slug without a redirect.
4. Upload an accurate image with useful alt text.
5. Confirm status and dates.
6. Inspect the page source or a structured-data validator. Never add ratings, offers, reviews, addresses, awards, or credentials unless verified.

Search and sitemap data use published repository queries only. Admin pages, API routes, drafts, and contact messages are excluded from indexing and search.
